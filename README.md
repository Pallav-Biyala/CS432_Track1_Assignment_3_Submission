# Mini Database System – ACID Validation (Module A)

## Overview
This module focuses on the ACID properties testing on our **B+ Tree–based mini database system** ,we created in previous assignment by introducing **transaction management** and **recovery mechanisms** in order to validate the four fundamental **ACID properties** of database systems.
The goal of this module is not to build a full production database engine, but to demonstrate how core transactional guarantees are implemented using simplified mechanisms such as **Write‑Ahead Logging (WAL)**, transaction management, and controlled crash simulations.
